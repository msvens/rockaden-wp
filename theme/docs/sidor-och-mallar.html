Rockaden-temat är ett blocktema (FSE) som använder WordPress sidredigerare. Sidor kan organiseras hierarkiskt med förälder/barn-relationer för att skapa sektioner med sidnavigation.

The Rockaden theme is a block theme (FSE) that uses WordPress's site editor. Pages can be organized hierarchically with parent/child relationships to create sections with sidebar navigation.

Sidmallar

Page Templates

Standardsida

Default Page

Den vanliga sidmallen är en enkel innehållskolumn. Vill du ha sidomenyn (höger sida), lägg in blocket Sidebar Panel där du vill att den ska visas. Se "Sidomeny på vanliga sidor" i Inställningar för mer info.

The standard page template is a single content column. If you want the sidebar, insert the Sidebar Panel block where it should appear. See "Sidebar on regular Pages" in Settings for more info.

Sektionssida

Section Page

En tvåkolumnslayout med vänsternavigation och innehåll till höger. Navigeringen byggs automatiskt från sidans förälder/barn-hierarki. Perfekt för samlingar av relaterade sidor som "Om Rockaden" med undersidor.

A two-column layout with left sidebar navigation and content on the right. Navigation is built automatically from the page's parent/child hierarchy. Perfect for collections of related pages like "About Rockaden" with sub-pages.

Så här använder du: Skapa en föräldersida och lägg till undersidor. Välj mallen "Page (Section Nav)" i sidans inställningar. Alla barn-sidor får automatiskt navigering i vänsterkolumnen. How to use: Create a parent page and add child pages. Select the "Page (Section Nav)" template in the page settings. All child pages automatically get navigation in the left column.

Sidvisning

Page Display

Dölj sidtitel

Hide Page Title

Du kan dölja sidtiteln via metarutan Rockaden Display i sidredigeraren. Kryssa i "Hide page title" för sidor där titeln inte behövs (t.ex. startsidan).

You can hide the page title via the Rockaden Display meta box in the page editor. Check "Hide page title" for pages where the title isn't needed (e.g. the landing page).

Shop-rutnät

Shop Grid

Temat tillhandahåller blocket Shop Items (under kategorin Rockaden i blockinfogaren). Det visar shopartiklar som ett rutnät — varje artikel har titel, bild, pris (samt eventuellt reapris), valfri köp-länk, lagerstatus (I lager / Slut i lager) och fri text för hur man beställer. Artiklarna hanteras under Shop Items i adminmenyn.

The theme provides the Shop Items block (under the Rockaden category in the inserter). It shows shop items as a grid — each item has a title, image, price (plus optional sale price), an optional buy link, a stock status (In stock / Out of stock), and free-text ordering instructions. Items are managed under Shop Items in the admin menu.
