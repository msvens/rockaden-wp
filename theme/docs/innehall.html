Guiden beskriver hur du skapar och formaterar innehåll med Rockaden-temat.

This guide describes how to create and format content with the Rockaden theme.

Inlägg

Posts

Inlägg visas på startsidan i omvänd kronologisk ordning. Varje inlägg visar datum, författare, utdrag och en valfri utvald bild. Inläggslistningen skapas automatiskt av temat.

Posts appear on the front page in reverse chronological order. Each post shows date, author, excerpt, and an optional featured image. The post listing is created automatically by the theme.

Tips: Skriv alltid ett utdrag (excerpt) för dina inlägg. Det visas i inläggslistningen och ger läsarna en bra förhandsgranskning. Tip: Always write an excerpt for your posts. It's shown in the post listing and gives readers a good preview.

Sidebar-kort

Sidebar Cards

Sidomenyn (höger) kan visa kort med information. Korten konfigureras under Utseende → Rockaden i sektionen "Sidebar Cards". Varje kort kan vara av typ text eller bild.

The right sidebar can display information cards. Cards are configured under Appearance → Rockaden in the "Sidebar Cards" section. Each card can be of type text or image.

Textkort

Text Cards

Textkort har en titel, HTML-innehåll, och en valfri länk med etikett. Använd dessa för "Aktuellt"-meddelanden, kontaktinfo eller snabblänkar.

Text cards have a title, HTML content, and an optional link with label. Use these for announcements, contact info, or quick links.

Bildkort

Image Cards

Bildkort visar en bild med valfri länk. Aktivera "Full card image" för att ta bort padding och bakgrund — användbart för logotyper och sponsorbilder.

Image cards display an image with an optional link. Enable "Full card image" to remove padding and background — useful for logos and sponsor images.

Mönster

Patterns

Temat innehåller blockmönster (patterns) som du kan infoga på sidor. Sök efter "Rockaden" i mönsterbiblioteket:

The theme includes block patterns that you can insert on pages. Search for "Rockaden" in the pattern library:

  • Hero — stor rubrik med undertext, passar för startsidan— large heading with subtitle, suitable for the landing page
  • News Grid — rutnät med de senaste inläggen som kort— grid of latest posts as cards
  • Page with Sidebar — tvåkolumnslayout med sidebar— two-column layout with sidebar
  • Page Three Column — trekolumnslayout— three-column layout
