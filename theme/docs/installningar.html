Temainställningar nås via Utseende → Rockaden i adminmenyn.

Theme settings are accessed via Appearance → Rockaden in the admin menu.

Sidhuvud

Header

Stil

Style

Välj mellan Default (vit bakgrund) och Contrast (mörk bakgrund i ljust läge). Kontraststilen ger ett mer framträdande sidhuvud.

Choose between Default (white background) and Contrast (dark background in light mode). The contrast style gives a more prominent header.

Densitet

Density

Styr sidhuvudets höjd: Kompakt (48px), Normal (56px) eller Stor (64px). På mobila enheter används alltid kompakt höjd.

Controls header height: Compact (48px), Normal (56px), or Large (64px). On mobile devices, compact height is always used.

CTA-knapp

CTA Button

En framträdande knapp i sidhuvudet som alltid syns, även på mobila enheter. Konfigurera etiketten (svenska och engelska) och vilken sida den ska länka till. Lämna URL:en tom för att dölja knappen.

A prominent button in the header that is always visible, even on mobile. Configure the label (Swedish and English) and which page it should link to. Leave the URL empty to hide the button.

Navigation

Navigation

Huvudnavigation

Main Navigation

Länkarna som visas centrerat i sidhuvudet. Lägg till, ta bort eller ändra ordning med repeater-fälten. Varje länk har en svensk etikett, en valfri engelsk etikett, och en URL (välj en befintlig sida eller ange en egen URL).

The links displayed centered in the header. Add, remove, or reorder using the repeater fields. Each link has a Swedish label, an optional English label, and a URL (select an existing page or enter a custom URL).

Mer-menyn

More Menu

Ytterligare länkar som visas i dropdown-menyn "Mer" (desktop) och i sidopanelen (mobil). Används för sekundär navigation som "Om oss" och "Kontakt".

Additional links shown in the "More" dropdown (desktop) and the side drawer (mobile). Used for secondary navigation like "About" and "Contact".

Mörkt läge

Dark Mode

Mörkt läge aktiveras via växlaren i Mer-menyn. Inställningen sparas i webbläsaren (localStorage). Vid första besöket följer temat systemets inställning. Du kan dölja mörkt läge-växlaren i temainställningarna.

Dark mode is toggled via the switch in the More menu. The setting is saved in the browser (localStorage). On first visit, the theme follows the system preference. You can hide the dark mode toggle in theme settings.

Språkväxlare

Language Switcher

Språkväxlaren (SV/EN) visas i Mer-menyn och påverkar navigeringsetiketter och block som stöder översättning. Inställningen sparas i webbläsaren. Du kan dölja språkväxlaren i temainställningarna.

The language switcher (SV/EN) appears in the More menu and affects navigation labels and blocks that support translation. The setting is saved in the browser. You can hide the language switcher in theme settings.

Sidebar

Sidebar

Den högra sidomenyn kan aktiveras globalt: Inaktiverad, Bara startsidan, eller Alla sidor. Enskilda sidor kan överrida denna inställning via Rockaden Display-metarutan.

The right sidebar can be enabled globally: Disabled, Landing page only, or All pages. Individual pages can override this setting via the Rockaden Display meta box.
