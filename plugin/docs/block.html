Pluginet tillhandahåller ett antal Gutenberg-block som du hittar under kategorin Rockaden i blockinfogarens sökruta. Block som kalender, träningsgrupper och turneringar används både på syntetiska sidor som skapas av temat och kan släppas in på vilken sida som helst.

The plugin provides a number of Gutenberg blocks, found under the Rockaden category in the block inserter search. Blocks for calendar, training groups, and tournaments are used both on the theme's generated pages and can be dropped onto any page.

Kalender

Calendar

Visar en månadskalender med evenemang, med stöd för månads-, vecko- och dagvy. Återkommande evenemang expanderas automatiskt. Inloggade redaktörer kan skapa, redigera och ta bort evenemang direkt från blocket — se Kalender för detaljer.

Shows a monthly calendar with events, with month / week / day views. Recurring events are expanded automatically. Signed-in editors can create, edit, and delete events directly from the block — see Calendar for details.

Ratinglista

Ranking List

Visar SSF-ratinglista för klubbens medlemmar. Hämtar data från SSF:s API baserat på det konfigurerade klubb-ID:t (se Inställningar → Rockaden).

Shows SSF player ratings for club members. Fetches data from the SSF API using the configured club ID (see Settings → Rockaden).

Träningsgrupp

Training Group

Visar en enskild träningsgrupp: schema, beskrivning, deltagarlista och länk till eventuell turnering. Används automatiskt på gruppens detaljsida.

Shows a single training group: schedule, description, participant list, and a link to the linked tournament if any. Used automatically on the group detail page.

Träningsgrupper (översikt)

Training Groups (overview)

Visar alla aktiva träningsgrupper i två kolumner — juniorer till vänster, vuxna (och blandade grupper) till höger. På smala skärmar staplas kolumnerna.

Shows all active training groups in two columns — juniors on the left, adults (and mixed groups) on the right. On narrow viewports the columns stack.

Turnering

Tournament

Visar en enskild turnering: titel, status, kategori, schema (om kopplad till evenemang), länk till officiellt resultat och ställning. För SSF-baserade turneringar hämtas data direkt från member.schack.se. Används automatiskt på turneringens detaljsida.

Shows a single tournament: title, status, category, schedule (if linked to an event), Official result link, and standings. For SSF-backed tournaments the data is fetched from member.schack.se. Used automatically on the tournament's detail page.

Turneringar (översikt)

Tournaments (overview)

Visar alla turneringar som ett rutnät med statusbrickor, kategori och SSF-märkning. Lämplig att lägga in på sidan /tournaments/ (Turneringar).

Shows all tournaments as a grid with status pills, category, and SSF badge. Drop it into the /tournaments/ (Tournaments) page.

Ställning

Standings

Visar enbart ställningstabellen för en turnering. Användbart när du vill bädda in en specifik turnerings resultat på en sida som inte är turneringens egen.

Shows just the standings table for a tournament. Useful for embedding a specific tournament's results on a page other than the tournament's own page.

Bildkarusell

Image Carousel

Bildspel eller karusell med konfigurerbart bildformat, anpassning, bakgrund, valfri helskärm och automatisk växling. Se Bildkarusell för fullständiga inställningar.

Slider or multi-item carousel with configurable aspect ratio, image fit, backdrop, optional fullscreen, and autoplay. See Image Carousel for the full settings.

Senaste nytt

Latest News

Visar de senaste publicerade inläggen. Antal inlägg ställs in i blockinställningarna.

Shows the most recent published posts. The number of posts is configurable in block settings.

Kommande evenemang

Upcoming Events

Visar en kompakt lista över kommande kalenderevenemang. Återkommande evenemang räknas in.

Shows a compact list of upcoming calendar events, including recurring ones.

Dokumentation

Documentation

Blocket bakom den här sidan. Renderar de bundlade hjälpfilerna i en två-kolumns-layout med sidonavigering. Endast inloggade redaktörer kan se sidan.

The block that powers this page. Renders the bundled help files in a two-column layout with sidebar nav. Visible only to signed-in editors.
