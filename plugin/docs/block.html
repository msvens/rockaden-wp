Pluginet tillhandahåller fem Gutenberg-block som du hittar under kategorin Rockaden i blockinfogarens sökruta.

The plugin provides five Gutenberg blocks, found under the Rockaden category in the block inserter search.

Kalender

Calendar

Visar en månadskalender med evenemang. Stöder kategorifiltrering och navigering mellan månader. Användare kan klicka på ett evenemang för att se detaljer.

Displays a monthly calendar with events. Supports category filtering and navigation between months. Users can click an event to see details.

Tips: Placera kalenderblocket på en dedikerad sida (t.ex. /kalender) för bästa resultat. Tip: Place the calendar block on a dedicated page (e.g. /kalender) for best results.

Ratinglista

Ranking List

Visar SSF-ratinglista för klubbens medlemmar. Hämtar data från SSF:s API baserat på det konfigurerade klubb-ID:t (se Inställningar → Rockaden).

Shows SSF player ratings for club members. Fetches data from the SSF API using the configured club ID (see Settings → Rockaden).

Ställning

Standings

Visar turneringsställning från en träningsgrupps rundturnering. Välj vilken grupp som ska visas i blockinställningarna.

Displays tournament standings from a training group's round-robin sessions. Select which group to show in block settings.

Träningsgrupp

Training Group

Visar detaljer för en enskild träningsgrupp: schema, deltagare och kommande pass. Används automatiskt på gruppens detaljsida (/training-groups/gruppnamn/).

Shows details for a single training group: schedule, participants, and upcoming sessions. Used automatically on the group detail page (/training-groups/group-slug/).

Träningsgrupper

Training Groups

Visar ett rutnät med alla aktiva träningsgrupper. Varje kort visar namn, beskrivning, schema och antal deltagare. Klicka på ett kort för att gå till gruppens detaljsida.

Displays a grid of all active training groups. Each card shows name, description, schedule, and participant count. Click a card to go to the group's detail page.

Tips: Placera detta block på träningsöversiktssidan (t.ex. /training). Tip: Place this block on the training overview page (e.g. /training).
