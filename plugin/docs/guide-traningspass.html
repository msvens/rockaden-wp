Menu names below are shown in English. If your admin language is Swedish they appear translated (e.g. Training → Träning).

Den här guiden visar hur du registrerar ett träningspass för en grupp — det vill säga närvaro och anteckningar för ett enskilt tillfälle.

This guide shows how to record a training session for a group — that is, attendance and notes for a single occasion.

Steg 1: Öppna gruppen

Step 1: Open the group

Gå till Training i adminmenyn och klicka på gruppen. Klicka sedan på fliken Sessions.

Go to Training in the admin sidebar and click the group. Then click the Sessions tab.

Steg 2: Starta passet

Step 2: Start the session

Har gruppen ett återkommande schema visas en lista med datum — klicka på Start Session vid det datum passet gäller. Saknar gruppen schema klickar du på Start Session högst upp, så skapas ett pass för dagens datum. Du tas direkt till passets vy.

If the group has a recurring schedule, a list of dates is shown — click Start Session next to the date the session is for. If the group has no schedule, click Start Session at the top to create a session for today. You're taken straight to the session view.

Steg 3: Registrera närvaro

Step 3: Record attendance

Under rubriken Attendance finns en kryssruta för varje deltagare. Bocka i dem som var närvarande.

Under the Attendance heading there's a checkbox for each participant. Tick the ones who were present.

Steg 4: Anteckningar och spara

Step 4: Notes and save

Skriv eventuella anteckningar under Notes (t.ex. vad passet handlade om) och klicka på Save. Närvaro och anteckningar sparas tillsammans.

Write any notes under Notes (e.g. what the session covered) and click Save. Attendance and notes are saved together.

Tips: Passets datum, anteckningar och antal närvarande visas på gruppens publika sida under fliken Sessions. Deltagarnas namn visas dock aldrig för besökare om Show participants är avbockat för gruppen. Tip: The session's date, notes and attendee count show on the group's public page under the Sessions tab. Participant names are never shown to visitors if Show participants is unticked for the group.
