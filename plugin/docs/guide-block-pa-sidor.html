The block and menu names below are shown in English. If your admin language is Swedish they appear translated.

Plugin och tema tillhandahåller en uppsättning block som du kan lägga på vilken sida som helst — träningsgrupper, kalender, ratinglistor, ställningar, nyheter, shop med mera. Den här guiden visar hur du lägger till dem. För detaljerade inställningar per block, se referensen Blocks.

The plugin and theme provide a set of blocks you can drop on any page — training groups, the calendar, rating lists, standings, news, the shop and more. This guide shows how to add them. For detailed per-block settings, see the Blocks reference.

Så lägger du till ett block

How to add a block

Öppna sidan i redigeraren, klicka på + där du vill ha innehållet, sök efter blockets namn (alla ligger under kategorin Rockaden) och klicka för att lägga till det. Översiktsblock blir ofta snyggast med inställningen Wide eller Full width.

Open the page in the editor, click + where you want the content, search for the block's name (they're all under the Rockaden category) and click to add it. Overview blocks usually look best with the Wide or Full width setting.

Vilket block för vad

Which block for what

  • Training Groups (overview) — alla aktiva träningsgrupper (junior till vänster, vuxen till höger). Inga inställningar behövs.— all active training groups (junior on the left, adult on the right). No settings needed.
  • Training Group — en enskild grupp. Välj grupp i blockets inställningar.— a single group. Pick the group in the block settings.
  • Calendar — en månadskalender med alla händelser. Inga inställningar behövs.— a month calendar with all events. No settings needed.
  • Upcoming Events — kommande händelser. Ange hur många som ska visas.— upcoming events. Set how many to show.
  • Tournaments (overview) — turneringar; pågående visas överst och avslutade i en hopfällbar lista längst ner. Inga inställningar behövs.— tournaments; ongoing on top and completed in a collapsible list at the bottom. No settings needed.
  • Tournament / Standings — en enskild turnering respektive dess ställning. Välj turnering i inställningarna.— a single tournament and its standings. Pick the tournament in the settings.
  • Ranking List — klubbens ratinglista från SSF (kräver klubb-ID under Settings → Rockaden).— the club's rating list from SSF (requires the club ID under Settings → Rockaden).
  • Latest News — de senaste nyhetsinläggen. Ange hur många.— the latest news posts. Set how many.
  • Shop Items — varor som rutnät. Ange antal och layout.— shop items as a grid. Set the count and layout.
  • Image Carousel — ett bildspel eller en karusell. Välj bilder och alternativ.— an image slideshow or carousel. Pick images and options.
Tips: Översiktsblocken (Training Groups, Tournaments, Calendar) hämtar innehållet automatiskt — du behöver inte uppdatera sidan när du lägger till en ny grupp, turnering eller händelse. Tip: The overview blocks (Training Groups, Tournaments, Calendar) pull their content automatically — you don't need to edit the page when you add a new group, tournament or event.
