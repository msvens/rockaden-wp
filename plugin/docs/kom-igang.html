Rockaden Chess är ett WordPress-plugin som lägger till schackklubbs-funktioner: träningshantering, kalender med evenemang, SSF-integration och Gutenberg-block.

Rockaden Chess is a WordPress plugin that adds chess club features: training management, calendar events, SSF integration, and Gutenberg blocks.

Funktioner

Features

  • Träningssystem Training system — hantera grupper, deltagare, träningspass, rundturneringar och närvaro. — manage groups, participants, sessions, round-robin tournaments, and attendance.
  • Kalenderevenemang Calendar events — skapa engångsevenemang eller återkommande evenemang med kategorifiltrering. — create one-off or recurring events with category filtering.
  • SSF-integration SSF integration — hämta ratinglistor, rankningar och importera evenemang från Svenska Schackförbundet. — fetch ratings, rankings, and import events from the Swedish Chess Federation.
  • Gutenberg-block Gutenberg blocks — bädda in kalender, ställningar och träningsinfo på valfri sida. — embed calendars, standings, and training info on any page.

Administration

Administration

Pluginet lägger till följande i WordPress-administrationen:

The plugin adds the following to the WordPress admin:

  • Rockaden → Träning Rockaden → Training — hantera träningsgrupper, deltagare och träningspass. — manage training groups, participants, and sessions.
  • Evenemang Events — skapa och redigera kalenderevenemang. — create and edit calendar events.
  • Inställningar → Rockaden Settings → Rockaden — konfigurera SSF klubb-ID. — configure SSF club ID.

SSF-inställningar

SSF Settings

Under Inställningar → Rockaden anger du klubbens ID på member.schack.se. Detta krävs för ratinglistan och SSF-eventimport. Hitta ID:t i URL:en när du visar din klubb på SSF:s webbplats.

Under Settings → Rockaden, enter your club's ID on member.schack.se. This is required for the ranking list block and SSF event import. Find the ID in the URL when viewing your club on the SSF website.
