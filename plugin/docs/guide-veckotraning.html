Menu names below are shown in English. If your admin language is Swedish they appear translated (e.g. Training → Träning).

Den här guiden visar steg för steg hur du skapar en träningsgrupp som träffas varje vecka, och kopplar en turnering till den så att ställning och resultat kan visas på gruppens sida.

This guide shows step by step how to create a training group that meets every week, and link a tournament to it so standings and results can show on the group's page.

Steg 1 (valfritt): Skapa turneringen först

Step 1 (optional): Create the tournament first

Vill du visa ställning och resultat för gruppen? Skapa då turneringen först. Gå till Tournaments i adminmenyn och klicka på Create Tournament. Fyll i namn, kategori och tidskontroll och spara. Låt statusen stå på Automatic och ange datumen, så flyttas den till "avslutad" av sig själv när slutdatumet passerat. Hoppa över det här steget om gruppen inte behöver någon turnering.

Want to show standings and results for the group? Create the tournament first. Go to Tournaments in the admin sidebar and click Create Tournament. Fill in name, category and time control, then save. Leave its status on Automatic and set the dates, so it moves itself to "finished" once the end date passes. Skip this step if the group doesn't need a tournament.

Steg 2: Skapa träningsgruppen

Step 2: Create the training group

Gå till Training i adminmenyn och klicka på Create Group. Fyll i:

Go to Training in the admin sidebar and click Create Group. Fill in:

  • Group Name — t.ex. ”Tisdagsträning vuxna”.— e.g. "Tuesday adult training".
  • Description, Trainers, Contact, Semester — alla valfria.— all optional.
  • Audience — Junior, Adult eller Mixed.— Junior, Adult or Mixed.
  • Status — normalt Automatic: härleds från schemats datum (planerad → pågående → avslutad). Du kan även välja manuellt eller Hidden (dold tills gruppen är klar).— normally Automatic: derived from the schedule's dates (planned → active → completed). You can also set it manually or to Hidden (until the group is ready).

Steg 3: Lägg till det återkommande schemat

Step 3: Add the weekly schedule

I samma dialog finns ett Event-avsnitt. Klicka på + New Event och fyll i start- och slutdatum med tid, bocka i Recurring och välj Weekly. Ange plats och välj kategorin Training. Schemat och platsen visas sedan automatiskt på gruppens sida.

The same dialog has an Event section. Click + New Event and fill in the start and end date with time, tick Recurring and choose Weekly. Set the location and pick the Training category. The schedule and location then show automatically on the group's page.

Steg 4: Koppla turneringen

Step 4: Link the tournament

Skapade du en turnering i steg 1? Välj den i listan Linked tournament. Spara sedan gruppen.

Did you create a tournament in step 1? Pick it in the Linked tournament dropdown. Then save the group.

Steg 5: Lägg till deltagare

Step 5: Add participants

Öppna gruppen och klicka på Add Participant. Sök bland klubbens medlemmar (hämtas från SSF) och klicka för att lägga till. Sökrutan stannar öppen så att du kan lägga till flera i rad.

Open the group and click Add Participant. Search the club's members (fetched from SSF) and click to add. The search box stays open so you can add several in a row.

Tips: Vill du ha gruppen synlig men hålla deltagarlistan privat (t.ex. för juniorer)? Bocka ur Show participants i gruppen — då döljs deltagarna för besökare, men schema och anteckningar visas fortfarande. Tip: Want the group public but the roster private (e.g. for juniors)? Untick Show participants on the group — visitors then won't see the participants, but the schedule and notes still show.
Ny termin? Öppna en befintlig grupp och klicka på Copy — då skapas en ny grupp med samma upplägg och deltagare; du ändrar bara namnet och anger de nya datumen. Turneringskopplingen följer inte med. New term? Open an existing group and click Copy — it creates a new group with the same setup and participants; you just change the name and set the new dates. The tournament link isn't carried over.
