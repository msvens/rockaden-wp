Menu names below are shown in English. If your admin language is Swedish they appear translated (e.g. Events → Evenemang).

Den här guiden visar hur du skapar en händelse som upprepas, t.ex. en träning varje vecka eller varannan vecka. Kalendern räknar automatiskt ut alla tillfällen åt dig.

This guide shows how to create an event that repeats, e.g. a training every week or every other week. The calendar automatically works out all the occurrences for you.

Steg 1: Skapa en ny händelse

Step 1: Create a new event

Gå till Events i adminmenyn och klicka på Add New.

Go to Events in the admin sidebar and click Add New.

Steg 2: Ange datum och tid

Step 2: Set date and time

Fyll i startdatum och slutdatum med tid. För en återkommande händelse anger du start- och sluttid för ett tillfälle — t.ex. 18:00–20:00.

Fill in the start date and end date with time. For a recurring event, set the start and end time of one occurrence — e.g. 18:00–20:00.

Steg 3: Gör den återkommande

Step 3: Make it recurring

Bocka i Recurring och välj Weekly eller Biweekly. Behöver du hoppa över enstaka datum (lov, helger)? Skriv in dem i Excluded Dates som kommaseparerade datum i formatet ÅÅÅÅ-MM-DD.

Tick Recurring and choose Weekly or Biweekly. Need to skip individual dates (holidays, breaks)? Enter them in Excluded Dates as comma-separated dates in YYYY-MM-DD format.

Steg 4: Detaljer och publicering

Step 4: Details and publishing

Ange plats, kategori, valfri länk och beskrivning, och klicka på Publish. Varje tillfälle visas som en egen post i kalendern.

Set the location, category, optional link and description, then click Publish. Each occurrence shows as its own entry in the calendar.

Tips: Slutdatumet styr hur länge serien pågår. Sätt det till sista tillfället i terminen så slutar serien där. Tip: The end date controls how long the series runs. Set it to the last occurrence of the term and the series stops there.
