Menu names below are shown in English. If your admin language is Swedish they appear translated (e.g. Events → Evenemang).

Den här guiden visar hur du lägger till en turnering som spelas via SSF (member.schack.se) i klubbens kalender, så att den syns bland övriga händelser.

This guide shows how to add a tournament that runs via SSF (member.schack.se) to the club calendar, so it appears among the other events.

Innan du börjar: klubb-ID

Before you start: club ID

Kontrollera att klubbens SSF-ID är ifyllt under Settings → Rockaden. Det behövs för att hämta data från SSF.

Make sure the club's SSF ID is set under Settings → Rockaden. It's needed to fetch data from SSF.

Steg 1: Skapa en ny händelse

Step 1: Create a new event

Gå till Events i adminmenyn och klicka på Add New.

Go to Events in the admin sidebar and click Add New.

Steg 2: Hämta turneringen från SSF

Step 2: Fetch the tournament from SSF

Längst upp i händelseformuläret finns avsnittet SSF Import. Skriv in turneringens SSF-ID och klicka på Fetch Tournament (eller Fetch Group om det gäller en gruppspelad serie). En förhandsvisning visar den hämtade datan så att du kan kontrollera att det blev rätt.

At the top of the event form is the SSF Import section. Enter the tournament's SSF ID and click Fetch Tournament (or Fetch Group for a group-played series). A preview shows the fetched data so you can confirm it's correct.

Steg 3: Fyll i datum och detaljer

Step 3: Fill in date and details

Ange startdatum och slutdatum med tid, en valfri plats, och välj kategorin Tournament. Du kan även lägga till en länk till turneringssidan och en beskrivning.

Set the start date and end date with time, an optional location, and pick the Tournament category. You can also add a link to the tournament page and a description.

Steg 4: Publicera

Step 4: Publish

Klicka på Publish. Händelsen dyker nu upp i kalendern (blocket Calendar) och bland kommande händelser.

Click Publish. The event now appears in the calendar (the Calendar block) and among the upcoming events.

Skillnaden mellan Fetch Group och Fetch Tournament: ”Group” avser en gruppspelad serie i SSF:s system, medan ”Tournament” avser en enskild turnering. Är du osäker, prova bägge och se vilken förhandsvisning som stämmer. Fetch Group vs Fetch Tournament: "Group" refers to a group-played series in SSF's system, while "Tournament" refers to a single tournament. If unsure, try both and see which preview matches.
