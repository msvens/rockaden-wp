Menu names below are shown in English. If your admin language is Swedish they appear translated (e.g. Tournaments → Turneringar).

Den här guiden visar hur du lägger en turnering som spelas via SSF (member.schack.se) i klubbens kalender, så att den syns bland övriga händelser. Sedan version 0.23 skapas kalenderkopplingen från själva turneringen — du behöver inte längre skapa en separat händelse.

This guide shows how to put a tournament that runs via SSF (member.schack.se) on the club calendar, so it appears among the other events. Since version 0.23 the calendar link is authored from the tournament itself — you no longer create a separate event.

Innan du börjar: klubb-ID

Before you start: club ID

Kontrollera att klubbens SSF-ID är ifyllt under Settings → Rockaden. Det behövs för att hämta data från SSF.

Make sure the club's SSF ID is set under Settings → Rockaden. It's needed to fetch data from SSF.

Steg 1: Skapa turneringen

Step 1: Create the tournament

Gå till Tournaments i adminmenyn och klicka på Create Tournament. Ange turneringens SSF-grupp-ID och klicka på Fetch / Refresh from SSF. Titel, datum, tidsformat och resultatlänk hämtas då från member.schack.se. Välj kategori och spara. (Mer om SSF-baserade turneringar finns i referensen Turneringar.)

Go to Tournaments in the admin sidebar and click Create Tournament. Enter the tournament's SSF Group ID and click Fetch / Refresh from SSF. Title, dates, time control and the result link are then pulled from member.schack.se. Pick a category and save. (See the Tournaments reference for more on SSF-backed tournaments.)

Steg 2: Lägg turneringen i kalendern

Step 2: Add the tournament to the calendar

Öppna turneringen och klicka på Edit. Kryssa i Add to calendar. Start- och slutdatum fylls i automatiskt från turneringens datum (de hämtade SSF-datumen). Ange en valfri plats, och om turneringen spelas återkommande (t.ex. varje vecka) kan du kryssa i Recurring och välja intervall. Kategorin är förvald till Tournament.

Open the tournament and click Edit. Tick Add to calendar. The start and end dates are filled in automatically from the tournament's dates (the fetched SSF dates). Set an optional location, and if the tournament runs on a recurring schedule (e.g. weekly) you can tick Recurring and choose the interval. The category defaults to Tournament.

Steg 3: Spara

Step 3: Save

Klicka på Save. Turneringen dyker nu upp i kalendern (blocket Calendar) på sina datum. Klickar en besökare på händelsen finns en länk vidare till turneringens egen sida.

Click Save. The tournament now appears in the calendar (the Calendar block) on its dates. When a visitor clicks the event, there's a link through to the tournament's own page.

Datumen följer SSF: Hämtar du turneringen på nytt från SSF och datumen har ändrats, uppdateras kalenderhändelsen automatiskt nästa gång du sparar turneringen. Du behöver inte röra händelsen separat. Dates follow SSF: If you re-fetch the tournament from SSF and the dates changed, the calendar event is updated automatically the next time you save the tournament. You never touch the event separately.
