Evenemang hanteras under Evenemang i adminmenyn. Varje evenemang har en titel, start- och slutdatum, valfri plats, länk, beskrivning och en kategori.

Events are managed under Events in the admin sidebar. Each event has a title, start/end date, optional location, link, description, and a category.

Kategorier

Categories

  • training — ordinarie träningspass— regular training sessions
  • tournament — klubb- eller externa turneringar— club or external tournaments
  • junior — junioraktiviteter— junior activities
  • allsvenskan — allsvenska matcher— league matches
  • meeting — styrelsemöten, årsmöten— board meetings, annual meetings
  • social — sociala evenemang— social events
  • other — övrigt— anything else

Återkommande evenemang

Recurring Events

Evenemang kan upprepas varje vecka eller varannan vecka. Ange upprepningsmönster och ett slutdatum. Pluginet expanderar automatiskt återkommande evenemang till enskilda förekomster i kalendern.

Events can recur weekly or biweekly. Set the recurrence pattern and an end date. The plugin automatically expands recurring events into individual occurrences in the calendar.

Undantagna datum: Du kan utesluta specifika datum från en återkommande serie (t.ex. helgdagar). Använd datumväljaren i evenemangseditorn för att lägga till undantag.

Excluded dates: You can exclude specific dates from a recurring series (e.g. holidays). Use the date picker in the event editor to add exclusions.

SSF-eventimport

SSF Event Import

När du redigerar ett evenemang kan du importera data från SSF:s kalender. Klicka på Importera SSF-evenemang i eventeditorns sidopanel för att söka och importera evenemang från member.schack.se. Importerad data fyller i titel, datum och länk.

When editing an event, you can import data from the SSF calendar. Click Import SSF Event in the event editor sidebar to search for and import events from member.schack.se. The imported data populates the event title, dates, and URL.
