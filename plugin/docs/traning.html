Träningshanteringen nås via Rockaden → Träning i adminmenyn. Den ger ett enhetligt gränssnitt för att hantera grupper, deltagare och träningspass.

The training manager is accessible from Rockaden → Training in the admin sidebar. It provides a unified interface for managing groups, participants, and sessions.

Grupper

Groups

En träningsgrupp representerar en återkommande aktivitet (t.ex. "Tisdagsträning vuxna"). Varje grupp har namn, valfri beskrivning, och kan kopplas till ett kalenderevenemang för schema och plats.

A training group represents a recurring activity (e.g. "Tuesday adult training"). Each group has a name, optional description, and can be linked to a calendar event for schedule and location.

Grupptyper

Group Types

  • TräningTraining — enbart träning, ingen turnering— training only, no tournament
  • TurneringTournament — enbart turnering (rundturnering)— tournament only (round-robin)
  • Träning och turneringTraining and tournament — kombination av träning och pågående turnering— combination of training and ongoing tournament

Hantera grupper

Managing Groups

  • Skapa:Create: Klicka Skapa grupp på träningssidan. Fyll i namn, schema och plats. Click Create Group on the training page. Fill in name, schedule, and location.
  • Redigera:Edit: Klicka på gruppens namn för att öppna detaljvyn, sedan Redigera. Click the group name to open the detail view, then Edit.
  • Ta bort:Delete: Klicka Ta bort i detaljvyn. Detta tar även bort alla träningspass. Click Delete in the detail view. This also removes all sessions.

Deltagare

Participants

Varje grupp har en deltagarlista. Deltagare kan kopplas till SSF-spelar-ID för automatisk ratinghämtning. Använd Lägg till deltagare i gruppens detaljvy.

Each group has a participant list. Participants can be linked to SSF player IDs for automatic rating lookups. Use Add Participant in the group detail view.

A session represents one meeting of a group. Each session can track:

  • NärvaroAttendance — markera vilka deltagare som var närvarande.— mark which participants were present.
  • PartierGames — registrera rundturneringslottning och resultat (Berger-metoden).— record round-robin pairings and results (Berger method).

SSF-koppling

SSF Integration

Du kan koppla en grupp till ett SSF-turneringsgrupp-ID för att hämta turneringsdata (lottningar, resultat, ställning) direkt från member.schack.se. Ange ID:t när du skapar eller redigerar gruppen och klicka Hämta från SSF.

You can link a group to an SSF Tournament Group ID to pull tournament data (pairings, results, standings) directly from member.schack.se. Enter the ID when creating or editing the group and click Fetch from SSF.
